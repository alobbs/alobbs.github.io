### Dark Elevator: Windows Install Service Local Privilege Escalation (CVE-2026-50343)

A local privilege escalation vulnerability (CVE-2026-50343) in Windows 11's InstallService allows a normal user to gain SYSTEM privileges by chaining two logic flaws: a writable plugin map registry key and a COM class whose backing DLL can be planted by any user under `%PROGRAMDATA%`. The exploit is deterministic, requires no administrator rights, UAC bypass, or reboot, and provides full SYSTEM-level code execution.

- [Dark Elevator: Windows Install Service Local Privilege Escalation (CVE-2026-50343)](https://blog.calif.io/p/dark-elevator-windows-install-service) — *blog.calif.io*

### log is non-monotonous in PHP and Lua

PHP and Lua's two-argument `log` functions use specialized base functions (like `log10`) for exact bases but compute `log(x)/log(base)` for others, creating a discontinuity at the boundary that can flip comparison results. ### Monte Kali

Monte Kali is a 520-meter-tall salt spoil heap in Heringen, Germany, created by K+S from potash mining byproducts, covering 98 hectares and containing about 201 million tonnes of salt as of 2016, with 900 tonnes added hourly. The heap has severely salinated the Werra river and groundwater, reducing invertebrate species from 60–100 to just 3, and dumping is licensed until 2030.
