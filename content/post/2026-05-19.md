## 🛡️ Security & Defense

### Microsoft promises to keep saved passwords out of Edge browser until needed

Microsoft is updating the Edge browser to prevent saved passwords from being loaded into memory in cleartext upon launch. This change addresses a security vulnerability that could allow attackers with administrative access to access user credentials. The update is rolling out across all supported versions of Edge and requires no action from users.

- [Microsoft promises to keep saved passwords out of Edge browser until needed](https://cybernews.com/privacy/microsoft-passwords-edge-browser/) — *cybernews.com*
